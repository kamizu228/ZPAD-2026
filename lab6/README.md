# Лабораторна робота №6
## C++ OpenCV

## Опис
Програма зчитує відео з камери, відображає його та дозволяє застосовувати різні режими обробки зображення за допомогою клавіатури.

## Функціонал
- Зчитування відео з камери
- Відображення відео у реальному часі
- Перемикання режимів обробки:
  - Оригінал
  - Інверсія кольорів
  - Gaussian Blur
  - Canny edge detection
- Інтерактивна взаємодія мишкою:
  - Ліва кнопка миші — малювання та збереження прямокутних маркерів на відео.
  - Права кнопка миші — очищення екрана від намальованих маркерів.
  - Коліщатко миші — плавне цифрове масштабування (Zoom) кадру.
- UI елементи:
  - Інтерактивний повзунок (Trackbar) для налаштування яскравості зображення у реальному часі.
  - Відображення лічильника FPS (кадрів за секунду) поверх відео.

## Вимоги
- Linux (Ubuntu)
- Камера
- C++ компілятор (gcc/g++)
- CMake
- OpenCV

## Встановлення та запуск

```bash
sudo chmod +x preinstall.sh build.sh run.sh

./preinstall.sh
./build.sh
./run.sh
```

## Керування
* q — вихід
* 1 — нормальний режим
* 2 — інверсія
* 3 — blur
* 4 — canny
* Ліва кнопка миші — малювання червоних прямокутників (зберігаються як зелені).
* Права кнопка миші — очищення екрана від усіх намальованих прямокутників.
* Коліщатко миші — плавний зум кадру (наближення/віддалення).
* Повзунок для зміни яскравості.
