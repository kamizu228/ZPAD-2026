# Лабораторна робота №7
## Computer Vision та багатопотоковість у C++

## Опис
Програма зчитує відео з вебкамери та виконує детекцію облич за допомогою нейромережі ResNet-10 через OpenCV DNN.

Детекція облич винесена в окремий потік (`std::thread`), що дозволяє зберігати плавне відео без блокування інтерфейсу.

## Функціонал
- Зчитування відео з камери
- Відображення відео у реальному часі
- Детекція облич
- Малювання рамки навколо облич
- Багатопотокова обробка кадрів
- Асинхронне оновлення результатів детекції

## Архітектура
### WebcamManager
Відповідає за отримання кадрів з вебкамери.

### Display
Відображає кадри у вікні OpenCV.

### KeyProcessor
Обробляє натискання клавіш та перемикання режимів.

### FrameProcessor
Виконує обробку кадрів.

### FaceDetector
Працює в окремому потоці:
- отримує кадри
- запускає нейромережу
- знаходить обличчя
- повертає координати знайдених облич

## Вимоги
- Linux (Ubuntu)
- Вебкамера
- CMake
- GCC / G++
- OpenCV
- OpenCV DNN module

## Встановлення

```bash
chmod +x preinstall.sh build.sh run.sh

./preinstall.sh
./build.sh
```

## Запуск

```bash
./run.sh
```

## Керування

- 1 — Звичайний режим
