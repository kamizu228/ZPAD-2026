# Лабораторна робота №4: 
## Візуалізація даних 2

## Основні етапи роботи
- Реалізація функції harmonic_with_noise для генерації гармонічного сигналу з шумом
- Додавання параметрів:
  - amplitude (амплітуда гармоніки)
  - frequency (частота гармоніки)
  - phase (фазовий зсув гармоніки)
  - noise_mean (середнє значення шуму)
  - noise_covariance (дисперсія шуму)
  - show_noise (прапорець, який визначає, чи потрібно показувати шум на графіку)
- Реалізація графічного інтерфейсу з елементами:
  - Поле для графіка функції
  - Слайдери для зміни параметрів гармоніки та шуму
  - Чекбокс для вмикання/вимикання шуму
  - Кнопка Reset для скидання параметрів
- Задання початкових значень для всіх параметрів гармоніки, шуму та фільтра
- Реалізація автоматичного оновлення графіка після зміни параметрів
- Реалізація логіки відображення:
  - якщо чекбокс вимкнений — показується «чиста» гармоніка
  - якщо чекбокс увімкнений — показується гармоніка з накладеним шумом
- Реалізація коректної зміни параметрів:
  - якщо змінюються параметри гармоніки, шум залишається тим самим і не генерується наново
  - якщо змінюються параметри шуму, змінюється лише шум, а гармоніка залишається незмінною
- Реалізація кнопки `Reset`, яка відновлює початкові значення параметрів
- Фільтрація зашумленої гармоніки за допомогою вибраного фільтра
- Налаштування параметрів фільтра через інтерактивні елементи інтерфейсу
- Відображення відфільтрованої гармоніки поряд з початковою для порівняння результатів
- Одночасне оновлення початкової та відфільтрованої гармонік після зміни параметрів
- Додавання інструкції для користувача щодо роботи з програмою

## Інструкція користувача
1. Запустіть програму.
2. За допомогою слайдерів змінюйте параметри гармоніки: амплітуду, частоту та фазу.
3. За допомогою окремих слайдерів змінюйте параметри шуму: середнє значення та дисперсію.
4. Використовуйте чекбокс для вмикання або вимикання відображення шуму.
5. Налаштовуйте параметри фільтра за допомогою відповідних елементів інтерфейсу.
6. Спостерігайте за початковою та відфільтрованою гармоніками на графіках.
7. Для повернення до початкових значень натисніть кнопку `Reset`.
