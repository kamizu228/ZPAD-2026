# Лабораторна робота №3
## Візуалізація даних

## Основні етапи роботи
- Вибір унікального багатовимірного датасету (Adult Census Income) із категоріальними та числовими атрибутами, а також пропусками
- Здійснення Data Cleaning (очищення від Missing Values та приведення типів даних)
- Побудова графіків залежності одного атрибута від іншого (scatter plot / lineplot)
- Побудова гістограми з розбиттям даних на 5 заданих діапазонів
- Багатовимірна візуалізація даних (відображення 3+ параметрів на одному графіку)
- Створення додаткових інформативних графіків (загальна кількість 5-8 штук)
