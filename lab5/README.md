# Лабораторна робота №5
## Наука про дані: обмін результатами та початковий аналіз

## Основні етапи роботи
- Створення веб-додатку з використанням бібліотеки Streamlit
- Завантаження підготовлених даних із лабораторної роботи №2 (`vhi_data.csv`)
- Створення dropdown-списку для вибору часового ряду:
  - VCI
  - TCI
  - VHI
- Створення dropdown-списку для вибору області
- Додавання slider для вибору інтервалу тижнів
- Додавання slider для вибору інтервалу років
- Додавання кнопки `Reset` для скидання всіх фільтрів до початкових значень
- Додавання двох checkbox для сортування даних:
  - за зростанням
  - за спаданням
- Реалізація реакції програми у випадку одночасного ввімкнення обох checkbox
- Розміщення інтерактивних елементів в одній колонці, а результатів аналізу — в іншій
- Створення трьох вкладок для відображення:
  - таблиці з відфільтрованими даними
  - графіка часового ряду для вибраної області
  - графіка порівняння вибраної області з іншими областями
- Реалізація фільтрації даних за областю, діапазоном тижнів і діапазоном років
- Побудова лінійного графіка для вибраного показника
- Побудова стовпчикового графіка для порівняння значень по областях

## Отримання даних (vhi_data.csv)
Файл `vhi_data.csv` формується в лабораторній роботі №2.

Для його отримання необхідно:
1. Відкрити файл `part1.ipynb` у лабораторній роботі №2.
2. Перейти до етапу Data Cleaning.
3. Після обробки даних (коли створено очищений DataFrame `df_clean`) додати рядок:

```python
df_clean.to_csv("vhi_data.csv", index=False)
```
4. Запустити notebook.
5. Після виконання з’явиться файл `vhi_data.csv`.

## Інструкція користувача
1. Скопіюйте отриманий файл `vhi_data.csv` у папку (поруч із lab5.py, README.md та requirements.txt).
2. Активуйте віртуальне середовище.
3. Встановіть необхідні залежності з файлу `requirements.txt`.
4. Запустіть Streamlit-додаток.
5. У лівій колонці оберіть:
   - показник (`VCI`, `TCI` або `VHI`)
   - область
   - інтервал тижнів
   - інтервал років
6. За потреби увімкніть сортування за зростанням або спаданням.
7. Натисніть `Reset`, щоб повернути всі фільтри до початкового стану.
8. Переглядайте результати у правій частині сторінки у вкладках:
   - **Таблиця**
   - **Графік**
   - **Порівняння**
